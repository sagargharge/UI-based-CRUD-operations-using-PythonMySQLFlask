##################################################################################################
				Read before you run the project
##################################################################################################
!. you must have any of the databases "MySQL or MariaDB", then you are supposed to create a new database in it  named "employee"

2. make sure you insert correct username('root' as obvious)  and password in your file 'run.py'

3. the project folder contains virtual environment, you have to activate the virtual environment before running the project.
   you can activate it by navigating to your project folder through your command line an entering the command 'source ./bin/activate'
   if it throws an error saying "no virtual environment installed", then install it with the help of the command 'sudo pip install virtualenv' or 'pip install virtualenv' and then activate it with the help of the above given command.

4. after activating the virtual environment run the project with 'python run.py'.

5. you will see the server started at your localhost, copy that address to your browser and see the project running.

6. now you can Create Retrieve Update & Delete records from your database.
